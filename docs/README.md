HEA-DRL 实验框架概述

本项目实现了一个由深度强化学习 (DRL) 引导的混合进化算法 (HEA) 框架，用于解决复杂调度/路径优化问题。实验设计包含 E1–E5 五组实验，对比不同移除策略 (随机移除、Shaw 移除、DRL 移除) 对算法性能的影响，并评估算法的效率和可扩展性。本文档集详细说明项目的架构、模块接口、数据配置、实验步骤以及结果产出，帮助开发者迅速理解设计思路和使用方法。

仓库结构

项目采用模块化结构，主要目录和文件如下：

src/：源代码目录，包含主要算法和模块实现：

hea.py：HEA 主算法流程实现

drl_agent.py：DRL 深度强化学习代理模块

removal.py：移除策略接口及其实现 (随机、Shaw、DRL)

repair.py：修复(重插入)策略实现

...其他源码文件...

data/：数据目录，包含问题实例数据集或实例生成脚本

configs/：配置文件目录，每个实验对应一个配置文件 (如 E1.yaml 至 E5.yaml)，定义实验参数

models/：模型目录，保存训练得到的DRL模型参数文件 (如 drl_model.pth)

results/：结果输出目录，各实验的日志和图表会保存在此处 (按实验编号子目录分类)

docs/：文档目录，即本说明文档集的各 Markdown 文件

快速开始

本项目使用 Python 3 环境开发，运行前请确保安装必要的依赖库（如 NumPy、PyTorch 等）。以下是快速运行一个实验的步骤：

安装依赖：使用 pip install -r requirements.txt（如果提供了依赖列表）。

准备数据：默认会自动生成随机数据，如需使用特定数据集请将其放置于 data/ 目录并在配置文件中指明路径。

运行实验：执行命令 python main.py --config configs/E1.yaml 可运行实验 E1 (使用随机移除策略的基准)。其他实验只需将参数文件替换为相应的 E2.yaml 等。

查看结果：程序完成后，在 results/E1/ 目录下查看日志文件和输出图表（详见 Results.md）。

更多运行命令和扩展功能请参见 Usage_Extensibility.md。如遇到问题，可查阅相应文档章节以获取详尽指导。
